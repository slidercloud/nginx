Docker NGINX

Index.html is the default file served when a request to port :8000 is made. However, you can also serve files like MP4 videos, images, etc..

By default, it is served using unsecure HTTP protocol. You can add SSL to your NGINX server using a free certificate from Let's Encrypt.

We highly recommend adding SSL certificate before deploying to production. If you deploy this site on Cloutty Edge Delivery, we'll provide you with a free SSL certificate.

Each time you add new files you'll have to rebuild and restart the docker container.

The same applies to Cloudflared tunnels.
